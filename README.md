## Project Overview 

A professional 6-page website for Mdukazi Projects, a Soweto-based ICT company 

established in 2006. The site showcases enterprise ITC services and solutions. 

 

## Website Goals 

- Establish strong digital presence for Mdukazi Projects 

- Showcase services: DIA, CCTV, Fibre, LAN, VPN, IT Support, NMS 

- Generate business leads via the contact form 

- Provide downloadable resource guides
